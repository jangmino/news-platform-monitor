

####
과제자문 주제: 플랫폼 산업의 자동화된 모니터링을 위한 프로그램 구축

□ 연구과제 세부자문내용
1. 연구자문 목적 및 내용
 ◦ AI 에이전트 기반 플랫폼 산업 자동 모니터링 시스템을 설계·개발하여, KISDI 디지털플랫폼 정책포럼의 데이터 기반 의제 설정 및 운영 효율화를 지원
  －주요 언론 뉴스(네이버 뉴스, 전문지 등) 및 정부·기관 공식 발표(보도자료, 의안, 기업 공지 등)를 자동 수집하는 크롤링 파이프라인 구축
  －수집된 데이터에 대해 핵심 키워드 추출, 긍정/부정 감성 분석, 정책 프레임 자동 분류(공정거래, 소비자보호, 개인정보, 노동, 콘텐츠/저작권, 안전, AI/자동화 등 7대 영역) 수행
 ◦분석 결과를 기반으로 리스크 스코어링 및 조기경보 체계를 구현하여, 포럼 위원 및 연구진이 즉시 활용 가능한 자동 리포트 생성
  －급상승 이슈 탐지(언급량 급증, 규제기관 동반 언급, 여론 급변 등 이상 징후 감지)
  －주간 이슈 브리핑(TOP 10 이슈 요약, 근거 링크, 정책 질문 자동 도출) 및 리스크 히트맵(플랫폼×정책영역 매트릭스) 자동 생성
2. 수행방법
 ◦생성형 AI(LLM) API 호출 기반의 에이전트 시스템으로 개발하며, KISDI 관계자 랩탑(Windows 환경)에서 직접 구동 가능하도록 설계
  － Python 기반 데이터 수집·분석 파이프라인 구축 (RSS 모니터링, 웹 크롤링, API 연동)
  － LLM 멀티라벨 분류를 활용한 정책 태깅 및 중복 제거·이슈 클러스터링 자동화
 ◦전체 처리 흐름은 수집 → 중복 제거/클러스터링 → 정책 태깅 → 리스크 점수 산출 → 브리핑 생성의 순서로 구성함
  －AI 분석 결과의 오류 가능성을 고려하여, 리스크 점수가 일정 수준 이상인 이슈는 사람이 확인하는 단계를 둠
  －요약문에는 반드시 원문 링크를 첨부하여 출처를 확인할 수 있도록 하고, 원문에 없는 내용이 포함되지 않도록 주의함
