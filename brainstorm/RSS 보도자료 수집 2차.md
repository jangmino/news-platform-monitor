# RSS 보도자료 수집

### 영역별 기관 선정

| **영역** | **대표 기관** |
| --- | --- |
| **공정거래** | **공정거래위원회** |
| **소비자보호** | **식품의약품안전처** |
| **개인정보** | **개인정보보호위원회** |
| **노동** | **고용노동부** |
| **콘텐츠/저작권** | **문화체육관광부** |
| **안전** | **행정안전부** |
| **AI/자동화** | **과학기술정보통신부** |

### 수집한 데이터 예시

- RSS 피드가 제공하는 데이터 개수가 최신 50개로 제한되어 있어, 부처별 보도자료 게시 빈도에 따라 수집 기간에 차이가 발생
    - 데이터 추가 수집 방안) 
    - 현재 7개 부처 외에 타 부처 RSS 추가
    - 대한민국 정책브리핑(korea.kr)의 보도자료 페이지 직접 크롤링
